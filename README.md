Open News operates on a **daily credit system** during the beta phase:

| Action | Credit Cost |
| :--- | :--- |
| AI Chat search | 1 credit + 1 per source fetched |
| Breaking News refresh | 1 credit per article |
| Developing refresh | 1 credit per article |

> **Daily Limit:** 100 credits — resets at midnight UTC.  
> **Beta Notice:** No payment system is active. We're actively working to expand access — please tune in later for updates!

- **Visualization**: [Recharts](https://recharts.org/)
- **Markdown Rendering**: [react-markdown](https://github.com/remarkjs/react-markdown)
- **Icons & Assets**: [Lucide React](https://lucide.dev/), [html-to-image](https://github.com/tsayen/html-to-image)

## <svg width="22" height="22" viewBox="0 0 24 24" fill="none" stroke="#ffd166" stroke-width="2" stroke-linecap="round" stroke-linejoin="round" style="vertical-align:middle;margin-right:6px;"><polygon points="13 2 3 14 12 14 11 22 21 10 12 10 13 2"/></svg> Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/Dattebayoolo/Open-News.git
cd Open-News

# 2. Install dependencies
npm install

# 3. Configure Environment Variables (.env)
# VITE_OPENROUTER_API_KEY=your_openrouter_api_key
# VITE_TAVILY_API_KEY=your_tavily_api_key

# 4. Run Development Server
npm run dev
```

### <svg width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="#ff4d4d" stroke-width="2" stroke-linecap="round" stroke-linejoin="round" style="vertical-align:middle;margin-right:4px;"><path d="M10.29 3.86L1.82 18a2 2 0 0 0 1.71 3h16.94a2 2 0 0 0 1.71-3L13.71 3.86a2 2 0 0 0-3.42 0z"/><line x1="12" y1="9" x2="12" y2="13"/><line x1="12" y1="17" x2="12.01" y2="17"/></svg> API Key Security

- **Never commit your `.env` file.** It is already listed in `.gitignore` by default.
- **Rotate keys regularly.** If you suspect a key has been exposed, revoke it immediately from the provider dashboard and generate a new one.
- **Avoid hardcoding keys** in source files, test scripts, or client-side bundles outside of Vite's `import.meta.env` pattern.
- **`test-openrouter.js`** is a standalone development/testing script that contains a hardcoded key example. It is **excluded from production** (ignored in `.gitignore` and removed from Git tracking). Do not use it in deployed environments.
