# Contributing to Open News

First off, thank you for considering contributing to Open News! It's people like you that make Open News such a great tool.

## How Can I Contribute?

### Reporting Bugs
-   Check the [issue tracker](https://github.com/Dattebayoolo/Open-News/issues) to see if the bug has already been reported.
-   If you find a new bug, please open a new issue. Include as much detail as possible: steps to reproduce, what you expected to see, and what you actually saw.

### Suggesting Enhancements
-   Open an issue with the tag "enhancement".
-   Explain why the enhancement would be useful and how it should work.

### Pull Requests
-   Fill in the pull request template (if available).
-   Ensure your code follows the existing style.
-   Make sure all tests pass (if applicable).
-   Update the documentation if you're adding new features.

## Style Guide
-   Use TypeScript for all new logic.
-   Follow the existing CSS patterns in `App.css`.
-   Keep components focused and reusable.

## Questions?
Feel free to open an issue or reach out to the maintainers if you have any questions!
